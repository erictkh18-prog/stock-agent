Account Management KB Content Management

Authentication Endpoints

Manage user accounts and admin approval workflows:

  • POST /auth/register - Register new user account
  • POST /auth/login - Authenticate and get JWT token
  • GET /auth/me - Get current user info
  • GET /auth/pending-users - List pending account approvals
  • POST /auth/approve/{email} - Approve account creation
  • POST /auth/reject/{email} - Reject account creation

Knowledge Base Endpoints

Manage knowledge base content contributions and approvals:

  • GET /admin/kb-chapters?status=Draft - List KB chapters by status
  • POST /knowledge-base/chapter-status - Approve/reject KB chapter
  • POST /knowledge-base/open-explorer - Open chapter file in explorer

Admin Dashboard Endpoint

Get comprehensive admin overview and metrics:

  • GET /admin/overview - Admin control-tower summary

Quick Reference

Authentication: Include JWT token in Authorization header: Authorization: Bearer <token>

Base URL: All endpoints are relative to your API host (e.g., http://localhost:8000)

Response Format: All endpoints return JSON responses

Interactive Docs: Full Swagger UI documentation available at /docs
